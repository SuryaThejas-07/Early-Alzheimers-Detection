# NeuroScan MRI - Project Descriptions

## 🎯 Primary Description (350 Characters)

**NeuroScan MRI** is an advanced deep learning system for early Alzheimer's detection using MRI scans. Built with PyTorch, it features a quantum attention-enhanced ResNet50 achieving 89.6% accuracy across four severity levels. The platform includes explainable AI (GradCAM visualization), an interactive Gradio web interface for diagnosis, and comprehensive PDF report generation. Designed for research and clinical support with test-time augmentation and class-balanced training.
